# Product Overview

HomeWizard Lite RESTFUL Proxy API is a Node.js-based proxy service that enables integration of HomeWizard Lite smart plugs with home automation systems like Home Assistant. 

The API acts as a bridge between the HomeWizard Lite cloud service and local home automation systems, providing RESTful endpoints to control smart plugs, dimmers, and curtains without relying on the official mobile app.

## Key Features
- Control smart plugs (on/off switching)
- Dimmer support with configurable minimum thresholds
- Curtain control (up/down/stop for Brel type curtains)
- Session management with caching
- Home Assistant compatibility
- Local state persistence using localStorage

## Target Users
Home automation enthusiasts who want to integrate cheap HomeWizard Lite devices into their existing home automation setups, particularly Home Assistant users.
